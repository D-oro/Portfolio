Simple things to build code for a fairer world

1. Use Firefox Dev Tools

Did you know Firefox Dev Tools have integrated accessibility testing tools?

  • Go to any webpage on Firefox
  • Right click
  • Select "Inspect Accessibility Properties"
    (the second option from below )

Firefox context menu

The accessibility inspector contains a lot of useful tools, including automated testing, a colour perception simulator, visually showing the tabbing order, and the accessibility tree of the whole page which can be printed to JSON.
I use the Firefox accessibility inspector most often for a quick way to check colour contrasts and to inspect elements in the accessibility tree.

Firefox Accessibility Inspector

Check out → the Firefox Accessibility Inspector documentation to learn more about this usefull tool.
